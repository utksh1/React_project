# 🏏 CriCode

A production-grade modern React + Tailwind dashboard inspired by cricket scoring and competitive coding platforms.

Track coding progress like cricket statistics, complete missions, unlock badges, analyze performance, and compete socially — all inside a clean aesthetic UI.

---

# ✨ Features

## 📊 Dashboard
- Hero overview section
- Daily streak tracking
- XP and runs statistics
- Match cards
- Recent activity feed
- Responsive modern layout

---

## 🏏 Matches
- T20 / ODI / Test match formats
- Match history cards
- Performance metrics
- Strike rates and score tracking
- Activity breakdown charts

---

## 🎯 Missions
- Daily missions
- Weekly challenges
- Special achievements
- Progress tracking bars
- Claim reward system

---

## 🏅 Badges
- Unlockable badge collection
- Common / Rare / Epic / Legendary categories
- Locked & unlocked badge states
- Achievement showcase

---

## 📈 Analytics
- Weekly performance charts
- Monthly growth tracking
- Skill assessment radar
- Difficulty distribution
- Productivity metrics

---

## 🌐 Social
- Global leaderboard
- Player rankings
- Community activity
- Future social features section

---

## 👤 Profile
- Career statistics
- Skill progress
- Season history
- Badge showcase
- Activity overview

---

# 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- React Router DOM
- Recharts
- Lucide React
